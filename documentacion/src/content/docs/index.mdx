## Características Principales

<CardGrid stagger>
	<Card title="Gestión de Procesos" icon="document">
		Administra todo el ciclo de vida de un proceso disciplinario: desde la apertura hasta el cierre, con estados automatizados y validaciones legales.
	</Card>
	<Card title="Inteligencia Artificial" icon="star">
		Integración con Google Gemini para generación automática de preguntas de descargos y análisis de respuestas con recomendaciones de sanciones.
	</Card>
	<Card title="Generación de Documentos" icon="list-format">
		Genera automáticamente citaciones, actas de descargos y documentos de sanción en formato PDF y Word con variables dinámicas.
	</Card>
	<Card title="Multi-tenant" icon="warning">
		Aislamiento completo de datos por empresa. Cada cliente solo puede ver y gestionar sus propios trabajadores y procesos.
	</Card>
	<Card title="Notificaciones y Seguimiento" icon="approve-check">
		Sistema de notificaciones en tiempo real con seguimiento de apertura de emails mediante pixel tracking.
	</Card>
	<Card title="Auditoría Completa" icon="seti:notebook">
		Timeline de cambios, trazabilidad de llamadas a IA y registro detallado de todas las acciones del sistema.
	</Card>
</CardGrid>

## Roles del Sistema

<CardGrid>
	<Card title="Administrador" icon="setting">
		Gestión completa del sistema: usuarios, empresas, catálogos y supervisión de todos los procesos.
		[Ver manual](/docs/manuales/administrador/)
	</Card>
	<Card title="Abogado" icon="open-book">
		Gestiona procesos disciplinarios, programa diligencias, genera preguntas con IA y emite sanciones.
		[Ver manual](/docs/manuales/abogado/)
	</Card>
	<Card title="Cliente (RRHH)" icon="approve-check">
		Registra trabajadores, solicita apertura de procesos y descarga documentos generados.
		[Ver manual](/docs/manuales/cliente/)
	</Card>
</CardGrid>

## Stack Tecnológico

- **Backend**: Laravel 12 + PHP 8.2+
- **Panel Admin**: FilamentPHP 3.2
- **Base de Datos**: MySQL 8.0+
- **Frontend**: Livewire 3 + Tailwind CSS 4 + Vite 7
- **IA**: Google Gemini API (gemini-2.5-flash)
- **Documentos**: dompdf + PHPWord
